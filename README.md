# Teyca Test — Панель управления программой лояльности


---

## ✨ Реализованный функционал

| Модуль | Статус | Описание |
|--------|--------|----------|
| 🔐 Авторизация | ✅ | Вход по логину/паролю, сохранение токена, защита маршрутов |
| 👥 Таблица клиентов | ✅ | Загрузка данных, пагинация, поиск, выбор чекбоксами |
| 🔍 Умный поиск | ✅ | По телефону (`7999...`), шаблону карты, баркоду |
| 📱 Отправка PUSH | ✅ | Модалка с валидацией, отложенная отправка, превью сообщения |
| 🎨 Интерфейс | ✅ | Angular Material, адаптивная вёрстка, обработка ошибок |

---
